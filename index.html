Junk King II

Diverte-te a reciclar o lixo de forma divertida e educativa

Como jogar

O jogo Junk King II é um jogo de reciclagem, onde o objetivo é reciclar o lixo de forma divertida e educativa.

Junk King II é um jogo feito para a internet com THREE.js com o tema "Eco-Escolas" e um pouco de inspiração do Junk King I. O jogo consiste numa "corrida" sem fim onde o jogador deve pegar um lixo e chegar ao final do nível. O nível é dividido em 2 partes; a primeira é uma mão gigante que só faz uma coisa, tentar impedir que o jogador progrida, para isso, a mão vai socar o chão repetidamente para que o caminho fique bloqueado. A segunda parte contém um parque de obstáculos, isto significa que o jogador deve pular de uma plataforma para outra, se o jogador cair, terá de recomeçar. Quando o jogador chegar ao final, deve escolher a cor correta do contentor para jogar o lixo, se o jogador escolher o contentor errado, terá de recomeçar. O jogo tem um sistema de pontuação, a pontuação é calculada pela quantidade de lixo que o jogador pega.
